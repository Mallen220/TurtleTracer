# Privacy Policy for Pedro Pathing Plus Visualizer

_Last Updated: 2026-02-10_

## Introduction

Pedro Pathing Plus Visualizer ("the Application") is an open-source desktop application designed to visualize and plan autonomous routines for FIRST Tech Challenge. We respect your privacy and are committed to protecting it.

This Privacy Policy explains what information the Application collects, how it is used, and your rights regarding your data.

## Data Collection and Usage

**We do not collect, store, or transmit any personal data.**

The Application is designed to operate locally on your device. Any data you create, import, or save within the Application (such as path files, robot configurations, or telemetry logs) remains on your local machine. We do not have access to this data.

## Third-Party Services

While the Application does not collect data directly, it utilizes third-party services for specific functionality:

### GitHub Releases (Updates)

The Application uses GitHub Releases to check for and download software updates. When the Application checks for an update, it makes a standard web request to GitHub's servers.

- **Data Involved:** GitHub (owned by Microsoft) may receive your IP address and other standard request headers as part of this process.
- **Purpose:** To provide you with the latest version of the software.
- **Privacy Policy:** Please refer to the [GitHub Privacy Statement](https://docs.github.com/en/site-policy/privacy-policies/github-privacy-statement) for more information on how they handle data.

## Local Storage

All application data, including settings, preferences, and project files, is stored locally on your computer's hard drive. You have full control over this data and can delete it at any time by removing the application or the associated files.

## Changes to This Privacy Policy

We may update our Privacy Policy from time to time. We will notify you of any changes by posting the new Privacy Policy on this page. You are advised to review this Privacy Policy periodically for any changes.
