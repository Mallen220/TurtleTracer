### What's New in 1.7.5!

## **Features:**

### **Workspace & UI Improvements**

- **Toggleable Coordinate System:** Switch seamlessly between Pedro and FTC coordinate conventions with a new toggle in the interface. This makes it significantly easier to work intuitively in your preferred reference frame without needing mental conversions.
- **Full Control over Path Deletion:** You can now delete the last remaining path segment using the on-screen delete button or keyboard shortcuts (Backspace/Delete). This allows for a completely empty workspace, which is perfect for starting fresh or clearing experimental paths.
- **Cleaner Starting Point UI:** To keep the interface focused and reduce screen clutter, the "Initial Heading" controls in the Starting Point section are now hidden when path segments already exist. They automatically reappear as soon as the path is cleared.
- **Visual Heading Indicator:** Angle input fields on the Paths tab now include a small, helpful heading indicator that shows the current orientation at a glance, making it much easier to visualize robot positioning.
- **Enhanced Ruler Readout:** The on-screen ruler now displays Delta X, Delta Y, and Angle in addition to straight-line distance, providing much better context and improved precision when measuring elements on the field.
- **Split Path Button:** Added a scissors button to the timeline playback controls. This makes splitting a path during timeline scrubbing easily discoverable without needing a keyboard shortcut.

### **App Experience & Performance**

- **Feedback & Rating Dialogs:** Share your thoughts easily with the new in-app feedback and rating system! Your direct input helps shape the future development of the Pedro Pathing Plus Visualizer.
- **Performance Optimization:** The `analyzePathSegment` routine has been optimized with adaptive sampling. This results in significantly faster path calculations and smoother graphical performance, especially when making adjustments on complex curves and longer paths.

## **Bug Fixes:**

### **Code Generation**

- **Sequential NextFTC Exporter:** Major fixes applied to the Sequential NextFTC code exporter to ensure reliable, correct output.
- **Auto-Export Consistency:** Auto-export now triggers correctly when project data changes (e.g. editing event markers) while enabled.

### **Animation Engine**

- **Empty Path Animation:** Fixed a crash that occurred when attempting to animate without any path lines.
- **Sequence Event Visualization:** Animation fixes: sequence events like "Rotate" now animate correctly even without a drawn path, and the robot is properly visualized in these scenarios.

### **General Stability**

- **Store Updates:** Fixed a critical issue where user settings were occasionally lost following a Microsoft Store update.
