### What's New!

## **Features:**
- **Drive Train Visualization:** Added a new setting to toggle between Holonomic, Swerve, and None for robot wheel visualization when using the default "No Image" robot.
- **Dynamic Wheel Speeds:** The default robot renderer now dynamically calculates and displays arrows in the corners corresponding to your drivetrain's wheel speeds and directions!
- **"No Image" Robot Mode:** You can now optionally use a lightweight placeholder (a green rectangle) instead of a robot image.
- **Heading Indicators:** Added a configurable heading arrow indicator for non-default robot images and the "no image" robot to easily see which way your robot is facing.

## **Bug Fixes:**
- **Playback Performance:** Fixed animation playback lag and improved time accuracy, especially on lower-end hardware.
- **Endpoint Headings:** Fixed an issue to ensure endpoint headings are preserved correctly.
- **Wheel Calculations:** Fixed various wheel velocity sign and motor output assignment bugs to ensure correct movement and rotation behavior in visualizations.
