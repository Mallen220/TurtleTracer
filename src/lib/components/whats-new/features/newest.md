### What's New!

## **Features:**
- **Split Path Button:** Added a scissors button to the timeline playback controls. This makes splitting a path during timeline scrubbing easily discoverable without needing a keyboard shortcut.

## **Improvements:**
- **Accessible Code Copying:** The "Copy Code" button in the Code Preview tab now provides better visual feedback when it is disabled (such as while generating code) and includes ARIA labels for improved screen reader support.
