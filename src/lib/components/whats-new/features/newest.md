# What's New in 2.0.1

With always new features coming out, we're excited to share the latest updates with you.

## **Onboarding Improvements**

We've refined the onboarding flow to ensure a smoother start for new users.

- **Dialog Awareness:** Onboarding steps now automatically pause whenever the Setup dialog is open, preventing UI overlaps and confusion.
- **Perfect Positioning:** Tutorial popovers have been adjusted for even better precision relative to their target elements.

## **Sidebar & Multi-Select**

Managing your workspace is now more powerful with new sidebar features.

- **Batch Actions:** Added support for shift-click multi-selection on the field paths, allowing you to perform actions on multiple items at once.
- **Unlink Macros:** You can now easily unlink macros from sidebar items directly, providing more flexibility in your workflow.
- **Improved Accessibility:** Comprehensive ARIA labeling has been added to the left sidebar for a better experience with screen readers.

## **Unified Previews**

Expanding on our preview system for more file types.

- **New Format Support:** Enjoy live previews for `Points`, `Turt`, and even your own `Custom` file formats, consistent with our existing path previews.

## **Workspace Refinements**

Several quality-of-life updates to the core field and tools.

- **Snap to Object:** The ruler tool now features "Snap to Object" functionality, making it easier to take precise measurements relative to your field elements.
- **Quick Deselect:** Clicking on any empty area of the field now conveniently deselects all active points.
- **Draw Path Mode:** New Draw Path function lets users sketch a path with a pencil tool and automatically generate a rough path from the drawing.
- **Keybind Enhancements:** Updated keybinds to allow deleting control points via `shift+d` and added `d` as a toggle for Draw tool mode.

## **Security & Maintenance**

Essential work under the hood to prepare for future releases.

- **Security Updates:** Applied security hardening patches and dependency updates.
- **TurtleTracer Branding:** All remaining references to the former project name have been removed in preparation for upcoming releases.
- **Backend Cleanup:** Extensive refactoring and cleanup of the backend codebase to improve performance and stability.
