# What's New in 2.1.0

This release is one of our biggest quality and capability updates yet, with major improvements across browser support, import/export workflows, simulation accuracy, file management, and overall UI polish.

## **Browser & Platform Expansion**

Turtle Tracer now works far better across environments.

- **Browser Support Added:** Core browser support has been added across the app.
- **File Manager in Browser:** Browser-compatible file manager behavior is now supported, improving parity with desktop workflows.
- **Browser Keybind Support:** Keyboard shortcuts now function in browser mode, improving productivity for web users.
- **macOS File Icon Fixes:** Fixed `.pp` and `.turt` icon display issues on macOS for better visual consistency.

## **Import, Export & Data Workflow**

Working with path data and robot assets is now more reliable and significantly more powerful.

- **Java Autonomous Importing:** Added Java autonomous import support.
- **AST-Based Parsing:** Importing now uses AST parsing to improve compatibility and handle a broader set of code structures.
- **Export Animation Regeneration:** Export animations are now force-regenerated when relevant settings change, preventing stale output.
- **Settings Synchronization:** Export Code Dialog settings are now synchronized with Code Tab settings to avoid conflicting configuration states.
- **Robot Profile JSON Export:** Added the ability to export robot profiles to JSON files for easier sharing and backup.

## **Simulation, Paths & Optimization**

Path behavior, heading handling, and optimization feedback are now more accurate and predictable.

- **Reverse Heading Simulation:** Improved reverse-heading simulation behavior for more realistic, less buggy, motion previews.
- **Motion Profile Scaling:** Improved simulation behavior for scaled motion profiles.
- **Facing Point Sequential Export Fix:** Fixed facing-point export behavior for sequential commands.
- **Heading Simulation Edge Fixes:** Resolved heading simulation issues that occurred in specific edge cases.
- **Optimization Dialog Overhaul:** Reworked the optimization dialog for clearer controls and better usability.
- **Immediate Optimizer Failure Feedback:** Returns `No valid path found` immediately when the optimizer cannot possibly succeed.

## **UI, Accessibility & Interaction Improvements**

Many parts of the interface were refined to reduce clutter and improve discoverability.

- **Accessibility Improvements:** Expanded accessibility enhancements across key UI areas.
- **File Manager UI Cleanup:** Improved file manager ergonomics and visual clarity.
- **Validate Button Redesign:** Replaced the larger Validate button with a smarter navbar icon treatment.
- **Timeline Marker Context Menu:** Added a small right-click context menu for timeline event markers.
- **Alignment & Distribution Menu:** Added context menu actions for aligning/distributing multiple selected points.
- **Scrollable Sidebar Icon Picker:** Added a scrollable icon picker menu for sidebar customization.
- **Grid Update:** Added `0.5"` grid size and removed axis labels for a cleaner field view.

## **Security, Stability & Refactors**

Under-the-hood hardening and maintainability work was a major focus in this version.

- **Macro Safety Protections:** Added extensive macro safety checks to prevent recursive/unsafe macro usage.
- **Plugin URL Protection:** Added URL protection safeguards against blind plugin behavior.
- **Security Improvements:** Applied additional security hardening updates.
- **Icon System Refactoring:** Completed broad icon refactors and major icon updates for consistency.
- **Performance Improvements:** Reduced overhead in key flows for a faster overall experience.
- **Edge Case Test Coverage:** Added stronger edge-case testing across critical features.
- **Additional Polish:** Includes many smaller fixes and quality-of-life improvements throughout the app.
