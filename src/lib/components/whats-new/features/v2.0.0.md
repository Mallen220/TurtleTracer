### What's New!

## **Important Changes:**

- **Renamed to Turtle Tracer:** The project has been renamed from **Pedro Pathing Plus Visualizer** to **Turtle Tracer** to better distinguish it from the original program and reflect its enhanced feature set.
- **New Project File Extension:** New saves now use `.turt`. Legacy `.pp` files are still supported and will be saved as `.turt` going forward.

### **UI & Navigation Enhancements**

- **Customizable Sidebar:** Added a new Left Sidebar that is highly scalable and fully customizable, putting all your essential tools right at your fingertips.
- **Optimized Layout:** The top Navbar has been minimized to give you a cleaner workspace, and the app now features better overall scalability on smaller devices.
- **Fresh Visuals:** A completely updated save icon, alongside a set of brand new icons across the interface for a modern feel.

### **Documentation & Help**

- **Web Documentation:** Removed the built-in documentation in favor of comprehensive, always up-to-date web documentation on our official website.
- **What's New Reimagined:** The "What's New" menu has been entirely overhauled (like the one you're looking at right now!) to clearly showcase the best new features in each update.

### **General Improvements**

- **Less Intrusive Rating:** The rating dialog has been adjusted so it will not open for at least five minutes after the application starts.
