# What's New in 2.1.2!

This release expands path-heading workflows, improves controls and accessibility, and continues core modernization across tooling and architecture.

## **Feature Additions & Workflow Enhancements**

Path planning and heading workflows are now more flexible and keyboard-friendly.

- **Piecewise Heading Added:** Added support for piecewise heading to improve control over segmented heading behavior.
- **Global Chain Heading Added:** Added a global chain heading option for consistent heading control across an entire chain.
- **New Chain/Heading Keybinds:** Added new keyboard shortcuts for chains, piecewise heading, and global chain heading actions.

## **Interface Controls & Accessibility Improvements**

Interaction controls are more predictable, and the interface is easier to use for more users.

- **Field View Lock Setting:** Added a setting to lock field view zoom and drag behavior.
- **Accessibility Upgrades:** Improved accessibility across key UI flows.
- **Stale Shortcut Fixes:** Fixed stale keyboard shortcuts so displayed mappings stay accurate.

## **Developer Experience & Quality Updates**

This release also strengthens project quality tracking and long-term maintainability.

- **Local Coverage Badge in README:** Added a local coverage badge to the README.
- **Codacy to SonarQube Cloud Migration:** Moved quality tracking from Codacy to SonarQube Cloud.
- **Large-Scale Code Cleanup:** Completed substantial cleanup work across the codebase.
- **More Svelte 4 to 5 Migration:** Continued the Svelte 4 to 5 migration effort in additional areas.
